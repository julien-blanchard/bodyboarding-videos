About

This is a small project that started in early 2026 on Eric L.'s Facebook page

Initially, the goal was to provide a comprehensive list of all the songs featured in Chris Stroh's Underground Tapes bodyboarding videos. Currently, 123 of the 155 total tracks are covered.

This project has now extended to covering more vintage bodyboarding videos.

Massive thanks to:

  • Eric L. for running the Underground Tapes Facebook page
  • Brent W. for his YouTube playlist
  • Chris R.H. for his help with the Headlifter tracks.
  • Julien S. for his help with the Mumma Green tracks.

Contribute

You are more than welcome to contribute to this project! Feel free to help me:

  • Identify any missing artist / song.
  • Find errors / typos within this webpage.
  • Improve the source code for this webpage. Currently the list of soundtracks sits in a Markdown file. I wrote a very simple script in TypeScript that reads this file and turns it into an html page (aka, this very website). The overall look and feel of the webpage is handled by Simple.css, and the whole thing is hosted on a Vercel free-tier account.
